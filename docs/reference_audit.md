## Design Rule

This project does not copy third-party skill content. Public skills and documentation are reviewed only to understand general patterns, including:

- folder organization
- YAML metadata structure
- trigger-oriented descriptions
- separation of instructions, templates, scripts, and references
- safety and validation sections

All academic workflow instructions, templates, scripts, and examples in this repository are written from scratch for this project.
