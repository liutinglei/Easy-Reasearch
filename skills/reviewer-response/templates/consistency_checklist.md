# Consistency Checklist

## Completeness

- [ ] Every editor, reviewer, examiner, or supervisor comment has a response.
- [ ] Multi-part comments are answered part by part.
- [ ] Minor formatting and language comments are not skipped.

## Accuracy

- [ ] Reviewer comments are quoted or summarized accurately.
- [ ] Claimed manuscript changes are present in the revised manuscript.
- [ ] Page, section, line, table, and figure references are verified.
- [ ] Citations, statistics, ethics details, and data statements are verified.

## Tone

- [ ] Responses are respectful and non-defensive.
- [ ] Disagreements are justified with evidence or scope limitations.
- [ ] The letter avoids exaggerated praise, emotional wording, and unsupported confidence.

## Submission Readiness

- [ ] The editor cover letter matches the response document.
- [ ] The response table and revision map use consistent comment IDs.
- [ ] All `[Author to verify]` items are resolved or intentionally retained.
- [ ] Clean and marked manuscript files match the response claims.
