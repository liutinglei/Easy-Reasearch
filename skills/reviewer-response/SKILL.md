# Reviewer Response

Use this skill to turn reviewer, editor, examiner, or supervisor feedback into a clear academic response package. Keep every response traceable to a comment, an action, a manuscript location, and a verification status.

## Operating Rules

- Preserve original comments separately from author responses.
- Answer every comment explicitly, including minor formatting comments.
- Separate manuscript changes from explanations made only in the response letter.
- Support both reviewer-specific and combined response letter modes.
- Use respectful, precise, non-defensive language.
- Mark missing locations, line numbers, references, or author decisions as `[Author to verify]`.
- Do not invent reviewer comments, manuscript changes, references, data, page numbers, line numbers, ethics details, or journal rules.
- Do not claim an issue is fully resolved unless the revised manuscript supports that claim.

## Workflow

1. Establish the review context: journal peer review, editor check, thesis examination, post-viva correction, or supervisor review.
2. Extract and number comments without changing their meaning.
3. Classify each issue: concept, literature, method, analysis, interpretation, limitation, language, formatting, citation, ethics, figure, table, or consistency.
4. Draft a point-by-point response with acknowledgement, direct answer, action taken, location, and status.
5. Build a revision map linking comments to manuscript changes.
6. Audit for overclaiming and unverifiable change claims.
7. Produce the requested package: editor letter, response table, revision map, consistency checklist, or final submission notes.

## Status Wording

- `addressed`: A manuscript change directly responds to the comment.
- `partially addressed`: The response addresses part of the concern or addresses it within stated scope limits.
- `clarified`: The response explains the issue or adjusts wording without a substantive new analysis.
- `not changed with justification`: No manuscript change was made, and the rationale is stated respectfully.
- `requires author verification`: The claim depends on unavailable manuscript text, line numbers, page numbers, data, or references.
