# Response Structure

## Standard Unit

Each response unit should include:

1. comment ID
2. preserved reviewer or examiner comment
3. response
4. revision made
5. manuscript location
6. status

## Modes

- `reviewer-specific`: group responses under each reviewer or examiner.
- `combined`: combine all comments into one table while preserving source labels.
- `editor-first`: answer editor requirements before reviewer comments.

Use reviewer-specific mode when reviewers supplied separate comment lists. Use combined mode for short technical checks or committee comments.
