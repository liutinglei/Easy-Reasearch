# Overclaiming Audit

Check every response claim against the manuscript.

## High-Risk Claims

- "fully resolved"
- "all inconsistencies"
- "new analysis"
- "substantially rewritten"
- "proved"
- "confirmed"
- "significant improvement"

## Safer Pattern

Use limited wording:

```text
We revised [specific location] to clarify [specific issue].
```

If the manuscript is unavailable, add `[Author to verify]`.
