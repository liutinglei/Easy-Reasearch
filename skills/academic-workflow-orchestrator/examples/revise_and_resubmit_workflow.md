# Revise-and-Resubmit Workflow

1. Collect editor decision letter, reviewer comments, revised manuscript, and journal instructions.
2. Use `reviewer-response` to create comment IDs, response table, and revision map.
3. Use `manuscript-audit` to verify response claims against manuscript changes.
4. Use `journal-submission-checker` to audit final package files and declarations.
5. Resolve all `[Author to verify]` items before submission.
