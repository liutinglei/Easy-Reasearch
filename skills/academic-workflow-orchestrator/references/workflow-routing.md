# Workflow Routing

Use focused skills for single-output tasks. Use the orchestrator when the task requires multiple stages, multiple documents, or a quality-gated package.

## Routing Examples

- Revise-and-resubmit: reviewer-response + manuscript-audit + journal-submission-checker
- Thesis correction: reviewer-response + manuscript-audit
- Systematic review project: systematic-review + manuscript-audit for final report checks
- Pre-submission audit: manuscript-audit + journal-submission-checker
