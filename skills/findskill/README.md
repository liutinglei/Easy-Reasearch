# findskill

`findskill` is a routing skill for this repository. It helps an agent choose the right academic workflow skill before doing substantive work.

Use it when a request is broad, ambiguous, multi-step, or could match several skills. The output should be a concise routing decision with missing inputs and uncertainty markers.
