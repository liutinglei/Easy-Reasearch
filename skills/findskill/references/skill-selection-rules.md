# Skill Selection Rules

Choose the narrowest skill that can complete the user request.

## Primary Signals

- Reviewer comments, response tables, rebuttal letters: `reviewer-response`
- Manuscript consistency, claims, abstract/results/discussion alignment: `manuscript-audit`
- Search strategies, eligibility criteria, screening, synthesis: `systematic-review`
- Cover letter, ethics, consent, data availability, submission package: `journal-submission-checker`
- Multi-stage project plan spanning multiple academic tasks: `academic-workflow-orchestrator`

## Missing Information

Ask or mark `needs user verification` when the task requires unavailable:

- manuscript text
- reviewer comments
- line or page numbers
- database search results
- journal instructions
- ethics approval details
- data availability details
